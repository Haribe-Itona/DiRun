# DiRun
DiRun - это ранер, ты бежиш вперед избегая препятствий, тем дальше пробежиш тем лучше. По пути противники и бафы. На этих забегах игрок может выбить ключи. Ключи используються что бы крутить лотерею в который выпадает босс. Боссы нужны что бы с ними драться

---- 1 стадия ----
1 Персонаж
•	Имеет 1 слот способности который можно заменить
•	100 единиц здоровья
•	слева 2 квадратные кнопки вперед и назад, 
•	справа в кнопка прыжка и способности.

2 камера
•	Камера постоянно движется направо
•	Игрок всегда на середине высоты камеры
•	Ускоряет свое движение если игрок переходит на правую половину экрана

3 меню
•	При первом нажатии Start запускается кат сцена которую можно скипнуть

4 Генерация ровней
•	Платформы спауняться случайно

 5 Паралакс Фон
 
6  Противники и препятсвия
•	Спауняться на платформах
•	Ходят туда сюда только по поверхности
•	Враги проходяться насквозь
•	Имеют атаку с анимацией

---- 2 стадия ----
1 Лут с мобов
2 Лотерея. для лотереи используются ключи 
С лотереи выбиваются босс файты с ДИ

3 Босс файты с ди
С каждого ди выпадает 1 предмет скин, способность …

4 слоты для скинов и пасивок персонажу
